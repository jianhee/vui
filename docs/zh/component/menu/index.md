# Menu 菜单

## 示例

### 基础用法

- 使用 `items` 属性定义菜单项数组
- 点击菜单项时触发 `click` 事件

<preview path="./demos/basic.vue"></preview>

### 选中菜单

- 使用 `selectable` 属性开启选中功能，使用 `v-model:selectedKey` 属性绑定选中值
- 选中菜单项时触发 `select` 事件

<preview path="./demos/select.vue"></preview>
