::: details 使用方法
样式相关的属性可以方便的设置组件的某一个样式，并且可以用 CSS 变量替代，示例如下：

| 类型     | 格式                      | 示例                                         | 单位                                                  |
| -------- | ------------------------- | -------------------------------------------- | ----------------------------------------------------- |
| Vue 属性 | `属性名="string\|number"` | `<VIcon size="20px" size="20" :size="20" />` | 可以省略，不带单位时会自动补全一个常用的值，比如 `px` |
| CSS 变量 | `--vui-组件名-属性名: 值` | `.vui-icon { --vui-icon-size: 20px; }`       | 不能省略                                              |

:::
