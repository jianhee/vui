## 数据状态 API

### 属性

| 名称           | 说明           | 类型             | 默认值  |
| -------------- | -------------- | ---------------- | ------- |
| `loading`      | 是否是加载状态 | `boolean`        | `false` |
| `loadingProps` | 加载组件的属性 | `VLoading.props` |         |
| `emptyProps`   | 空组件的属性   | `VEmpty.props`   |         |

### 插槽

| 名称      | 说明                                         | 参数 |
| --------- | -------------------------------------------- | ---- |
| `loading` | 加载状态时显示的自定义内容，优先级高于属性值 |      |
| `empty`   | 空状态时显示的自定义内容，优先级高于属性值   |      |
