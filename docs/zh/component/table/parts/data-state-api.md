## 数据状态 API

### 属性

| 名称        | 说明                   | 类型      | 默认值    |
| ----------- | ---------------------- | --------- | --------- |
| `loading`   | 是否正在加载数据       | `boolean` | `false`   |
| `emptyText` | 空数据时显示的文本内容 | `string`  | `No Data` |

### 插槽

| 名称      | 说明                                       | 参数                                                                                   |
| --------- | ------------------------------------------ | -------------------------------------------------------------------------------------- |
| `default` | 节点的自定义内容，优先级高于属性值         | 表格 `{ row: '当前行', col: '当前列' }` <br> 树 `{ node: '当前节点', item: '当前项' }` |
| `empty`   | 空数据时显示的自定义内容，优先级高于属性值 |                                                                                        |
