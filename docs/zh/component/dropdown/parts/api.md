### 属性

| 名称                   | 说明                                     | 类型                                  | 默认值  |
| ---------------------- | ---------------------------------------- | ------------------------------------- | ------- |
| `trigger`              | 触发方式，右键对齐鼠标，其它对齐元素     | `string`: `hover\|click\|contextmenu` | `hover` |
| `closeOnClickDropdown` | 是否在点击下拉框时关闭，默认点击外部关闭 | `boolean`                             | `false` |

### 插槽

| 名称       | 说明                               | 参数 |
| ---------- | ---------------------------------- | ---- |
| `default`  | 触发器，必须是有效的 html DOM 元素 |      |
| `dropdown` | 下拉框内容                         |      |

### 事件

| 名称    | 说明       | 参数 |
| ------- | ---------- | ---- |
| `open`  | 打开时触发 |      |
| `close` | 关闭时触发 |      |

### 方法

| 名称      | 说明       | 参数                                                        |
| --------- | ---------- | ----------------------------------------------------------- |
| `open()`  | 打开下拉框 | `{ event: '事件对象', el: '参考元素，优先级高于事件对象' }` |
| `close()` | 关闭下拉框 |                                                             |
